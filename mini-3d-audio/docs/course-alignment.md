# Course Alignment — mini-3d-audio

## Nine-University Curriculum Mapping

| University | Course | Module Coverage |
|------------|--------|-----------------|
| **MIT** | 6.003 Signal Processing | Convolution (OLA), Fourier analysis (DFT→HRTF), window functions, sampling theory |
| **MIT** | 6.630 EM Waves | Spherical harmonics (same math as acoustic multipole expansion), wave reflection |
| **Stanford** | EE102A Signal Processing | Coordinate transforms, vector spaces, filter design |
| **Stanford** | EE359 Wireless | Spatial channel modeling (analogous to HRTF spatial filtering), diversity |
| **Berkeley** | EE123 DSP | Overlap-add convolution, FIR/IIR filters, FFT-based processing, window design |
| **Berkeley** | EE117 EM Waves | Image theory (room reflections ↔ EM boundary conditions) |
| **Illinois** | ECE310 DSP | Sampling theory, Nyquist, multi-rate processing (Doppler resampling) |
| **Michigan** | EECS351 DSP | Audio signal processing, filter bank design, time-varying filters |
| **Michigan** | EECS411 Microwave | Cavity resonance ↔ room modes, spherical mode theory |
| **Georgia Tech** | ECE4270 DSP | DSP system integration, real-time signal processing pipelines |
| **Georgia Tech** | ECE6350 EM | Spherical vector waves (same basis functions as Ambisonics) |
| **TU Munich** | Signal Processing | Acoustic signal processing, spatial audio algorithms |
| **TU Munich** | High-Frequency Engineering | Ray tracing ↔ image source method, multipole expansion |
| **ETH** | 227-0427 Signal Processing | Adaptive filtering, time-frequency analysis |
| **ETH** | 227-0455 EM Waves | Spherical wave expansion, Helmholtz equation on sphere |
| **Tsinghua** | Signal & Systems | Transform-domain processing, spectral estimation, window functions |
| **Tsinghua** | Communication Principles | Channel modeling, spatial diversity |

## Textbook Alignment

| Textbook | Author(s) | Module Usage |
|----------|-----------|--------------|
| Signals and Systems | Oppenheim & Willsky (1997) | Convolution, Fourier analysis foundations |
| Discrete-Time Signal Processing | Oppenheim & Schafer (2010) | Overlap-add, FFT, filter structures |
| Digital Communications | Proakis & Salehi (2008) | Spatial channel modeling |
| Fundamentals of Applied EM | Ulaby & Ravaioli (2015) | Spherical harmonics, wave propagation |
| Digital Video and HD | Poynton (2012) | Audio/video synchronization (foundation for AV sync) |
| Spatial Hearing | Blauert (1997) | Binaural cues, localization theory |
| HRTF and Virtual Auditory Display | Xie (2013) | HRTF measurement, interpolation, personalization |
| Ambisonics | Zotter & Frank (2019) | HOA encoding, decoding, NFC |
| Room Acoustics | Kuttruff (2009) | Sabine/Eyring, image source method |
