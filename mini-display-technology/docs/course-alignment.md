# Course Alignment — mini-display-technology

## Nine-School Mapping

| School | Course | Topics Covered Here |
|--------|--------|---------------------|
| **MIT** | 6.003 Signal Processing | Sampling theory (pixel clock bandwidth), Nyquist, transform domain (DCT/subsampling) |
| **MIT** | 6.450 Digital Comm | TMDS 8b/10b line coding, channel capacity (Shannon-Hartley for cables), scrambling |
| **Stanford** | EE102A Signal Processing | Sampling, interpolation (image scaling), quantization error analysis |
| **Stanford** | EE247 Optical/Display | Color science (CIE XYZ, Lab), display metrology, gamma calibration |
| **Berkeley** | EE123 DSP | 2D filtering (Gaussian, Sobel), histogram equalization, dithering (error diffusion) |
| **Berkeley** | EE117 EM Waves | Spectral power distribution × CMF, radiometry to photometry |
| **Illinois** | ECE 310 DSP | Multirate (image scaling as decimation/interpolation), 2D convolution |
| **Illinois** | ECE 459 Comm | Line coding (TMDS), link training, equalization |
| **Michigan** | EECS 351 DSP | Image processing (scaling, dithering, enhancement), filter design |
| **Georgia Tech** | ECE 4270 DSP | Image compression fundamentals, color subsampling |
| **Georgia Tech** | ECE 6350 EM | Signal integrity (eye diagram, insertion loss, cable attenuation) |
| **TU Munich** | Display Technology | VESA timing, panel types, backlight, luminance specification |
| **TU Munich** | High-Frequency Eng. | TMDS/DP high-speed link design, impedance control |
| **ETH** | 227-0427 Signal Processing | Filter banks (subband decomposition for video), sampling rate conversion |
| **ETH** | 227-0455 EM/Optics | Colorimetry, photometry, CIE standards |
| **Tsinghua** | 信号与系统 (Signals & Systems) | Fourier analysis (DCT for image compression), sampling |
| **Tsinghua** | 通信原理 (Communication Principles) | Source coding (pixel encoding), channel coding (TMDS) |
