# Course Tree — mini-streaming-protocol

## Prerequisites
- C programming (all implementations)
- Network fundamentals (TCP/UDP/IP)
- Binary/hex arithmetic (packet serialization)
- Basic signal processing (EWMA filters)
- XML text format (DASH MPD)

## Dependencies on mini-electronic-info modules
1. **mini-communication-principle** — modulation, coding, channel models → applied in QoS assessment
2. **mini-digital-signal-process** — filters, transforms → EWMA jitter filter, NTP conversion
3. **mini-wireless-mobile-comm** — wireless channel → ABR over mobile networks
4. **mini-audio-codec / mini-video-codec** — media encoding → payload formats

## Postrequisites
- **mini-av-sync** — audio-video synchronization over streaming
- **mini-iot-edge-computing** — edge streaming analytics
- Advanced: WebRTC, 5G NR streaming, cloud gaming protocols
